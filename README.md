# Posts

A collection of technical guides and posts covering various development topics.

## Contents

This repository contains the following documentation files:

### [MigrationStep.md](MigrationStep.md)
A guide on handling migrations in Optimizely CMS when renaming properties and content types. This post explains:
- How GUIDs protect against data loss when renaming content types
- When migration steps are necessary
- How to implement migration steps to preserve existing data
- Examples of renaming properties and content types

### [NodeVersionIssue.md](NodeVersionIssue.md)
A troubleshooting guide for resolving Node.js version compatibility issues. Topics covered:
- Common Node.js version errors (e.g., digital envelope routines errors)
- How to check and manage Node.js versions using nvm
- Steps to switch Node versions and reinstall dependencies
- Best practices for resolving build errors

### [SwitchGitUserPrFolder.md](SwitchGitUserPrFolder.md)
A guide on configuring different Git user identities per project folder on macOS. This post covers:
- Why you might need separate Git identities for personal and client projects
- How to set a local Git email for a specific project
- Steps to verify and manage Git user configuration
- Keeping personal and professional Git activity separate

### [VirtualAndOverride.md](VirtualAndOverride.md)
A practical guide on using `virtual` and `override` keywords in C#. The post demonstrates:
- How virtual methods work in base classes
- Overriding virtual methods in derived classes
- Real-world example using Excel worksheet header creation
- Benefits of using virtual/override for cleaner, more maintainable code

## Purpose

These posts serve as quick reference guides and documentation for common development scenarios and best practices.
